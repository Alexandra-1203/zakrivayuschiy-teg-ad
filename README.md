Сайт-дневник с рефлексией о прохождении курса по фронтенд-разработке. Каждый блок соответствует спринту обучения.

🔗 Ссылка: https://alexandra-1203.github.io/zakrivayuschiy-teg-ad/

Чему научилась в проекте:

1. Адаптивная верстка
   
   Медиазапросы для мобильных, планшетов и десктопов

   Резиновая сетка, правильный viewport

2. Анимации и интерактив

   Плавные переходы при наведении

   SVG-спрайты с изменением цвета

   Состояния :active, :focus

   Анимация и трансформация элементов

   Модальное окно на нативном dialog
   
3. Построение сеток

   Flexbox для выравнивания

   Grid для раскладки карточек

4. Организация кода

   CSS-переменные (цвета, шрифты, отступы)

   БЭМ-именование классов
   
   Кроссбраузерность (вендорные префиксы, fallback-значения)

5. Типографика и графика

   Подключение вариативных шрифтов

   Оптимизация изображений

   Адаптивные единицы (rem, %, vw/vh)
